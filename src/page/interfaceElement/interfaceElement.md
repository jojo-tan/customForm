# UI组件库 element-ui

# 总体构成
1. 多界面元素页-tab切换
2. 单个界面元素页由控件列表，元素列表，属性操作三个页面组成

# 功能要求
1. 控件列表页可拖拽相应控件至元素列表，元素列表新增对应的表单元素
2. 元素列表中表单元素拖拽重新排序
3. 元素列表表单元素可删除，修改（修改时属性操作页对应生成相应的修改信息）
4. 属性操作页修改的表单元素信息实时刷新元素列表页
5. 元素列表页可以提交对应界面元素页Json信息至后台
6. 元素列表页可以模拟提交表单信息进行表单验证

# 控件元素
1. 纯文本
2. 文本框
3. 单选按钮组
4. 复选按钮组
5. 下拉列表选项框
6. 日期选择
7. 扫一扫
